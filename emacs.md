
| Keys | Description | Action |
|------|-------------|--------|
| C | control key |  |
| M | meta key | |
| C-x C-x | close emacs | close emacs:w |
| C-g | quit | quit a partialy entered command |
| C-v | forward one screen | move forward one screen |
| M-v | back one screen | move back one screen |
| C-l | clear screen | move text where cursor is at to the center |
| C-p | previous | move cursor up |
| C-n | next | move cursor down |
| C-b | back | move cursor back |
| C-f | forward | move cursor forward |
| M-f | forward word | move forward one word |
| M-b | backward word | move back one word |
| C-a | begining of line | move to the begining of line |
| C-e | end of line | move to the end of line |
| M-a | begining of sentence | move to the beginign of sentence |
| M-e | end of sentence | move to the end of sentence |
| M-> | end of file | move to the end of the file |
| M-< | begining of file | move to the begining of file |
